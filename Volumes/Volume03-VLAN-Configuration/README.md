## About this Volume

This volume covers VLAN configuration, trunking concepts, and basic switching segmentation in Cisco networks.

VLANs are a fundamental concept in Layer 2 switching and are essential for network segmentation, security, and traffic isolation.

You will also learn how VLANs interact with access ports and trunk links, which are critical for multi-switch environments.

---

## What You Will Learn

- VLAN Creation and Management
- Access Port Configuration
- Trunk Port Configuration (802.1Q)
- Native VLAN Concepts
- VLAN Assignment to Ports
- VLAN Verification
- Inter-Switch VLAN Communication
- Basic Troubleshooting of VLAN Issues
