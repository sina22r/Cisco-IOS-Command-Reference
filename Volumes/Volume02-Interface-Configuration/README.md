# Interface Configuration

**Estimated Reading Time:** 60 Minutes

## About this Volume

This volume covers the configuration, verification, and troubleshooting of Cisco router and switch interfaces.

You will learn how to configure Layer 2 and Layer 3 interfaces, assign IP addresses, manage interface states, configure interface descriptions, and verify operational status.

The commands in this volume are commonly used in CCNA, CCNP ENCOR, and day-to-day network administration.

---

## What You Will Learn

- Interface Configuration
- Interface Administration
- IP Address Assignment
- Interface Descriptions
- Speed and Duplex Configuration
- MTU Configuration
- Interface Verification
- Loopback Interfaces
- Shutdown and Recovery
- Interface Troubleshooting Basics
