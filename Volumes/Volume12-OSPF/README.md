## About this Volume

This volume covers Open Shortest Path First (OSPF), one of
the most widely deployed Interior Gateway Protocols (IGPs)
in enterprise networks.

OSPF is a link-state routing protocol that uses the
Shortest Path First (SPF) algorithm to calculate optimal
paths and provides fast convergence, hierarchical routing,
and efficient scalability through the use of areas.

Both OSPFv2 (IPv4) and OSPFv3 (IPv6) are covered.
