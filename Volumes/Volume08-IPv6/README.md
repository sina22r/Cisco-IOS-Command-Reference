## About this Volume

This volume covers IPv6 configuration, verification, and troubleshooting using Cisco IOS.

IPv6 is the successor to IPv4 and provides a vastly expanded address space, simplified header format, built-in support for Neighbor Discovery, Stateless Address Autoconfiguration (SLAAC), and modern routing capabilities.
