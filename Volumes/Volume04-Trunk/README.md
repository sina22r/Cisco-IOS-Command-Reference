## About this Volume

This volume covers trunk configuration in Cisco IOS. Trunk links
allow multiple VLANs to travel across a single physical connection
between network devices using IEEE 802.1Q encapsulation.

Understanding trunking is essential for building scalable Layer 2
networks and connecting multiple switches together.
