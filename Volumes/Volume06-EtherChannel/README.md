## About this Volume

This volume covers EtherChannel configuration and verification in Cisco IOS.

EtherChannel combines multiple physical Ethernet links into a single logical interface to increase bandwidth, improve redundancy, and simplify network management. Cisco supports EtherChannel using both the proprietary Port Aggregation Protocol (PAgP) and the IEEE 802.3ad Link Aggregation Control Protocol (LACP).
