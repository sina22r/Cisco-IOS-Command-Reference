# Volume 09

# Static Routing

---

**Estimated Reading Time:** 90 Minutes

---

## About this Volume

This volume covers IPv4 and IPv6 static routing using Cisco IOS.

Static routes provide predictable forwarding behavior, lower
resource utilization, and are commonly used in enterprise
networks, WAN deployments, backup paths, and default routing.
