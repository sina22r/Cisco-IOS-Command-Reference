# Spanning Tree Protocol (STP)

---

**Estimated Reading Time:** 90 Minutes

---

## About this Volume

This volume covers Spanning Tree Protocol (STP), one of the
most important Layer 2 technologies in Cisco networks.

STP prevents switching loops by creating a loop-free logical
topology while maintaining redundant physical links for
fault tolerance.
