## About this Volume

This volume covers Border Gateway Protocol (BGP), the
standard Exterior Gateway Protocol (EGP) used to exchange
routing information between Autonomous Systems.

BGP is a path-vector routing protocol designed for scalability,
policy-based routing, and Internet-scale deployments. It is
widely used by Internet Service Providers (ISPs), cloud
providers, data centers, and enterprise edge networks.

Both Internal BGP (iBGP) and External BGP (eBGP) are covered.
