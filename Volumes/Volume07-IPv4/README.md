# IPv4

---

**Estimated Reading Time:** 90 Minutes

---

## About this Volume

This volume covers IPv4 addressing and interface configuration
using Cisco IOS.

IPv4 is the foundation of Layer 3 networking. This volume
includes commands for configuring IPv4 addresses, interface
parameters, routing behavior, and verification.

---

## What You Will Learn

- IPv4 Interface Configuration
- Secondary IP Addresses
- Interface MTU
- TCP MSS Adjustment
- Directed Broadcast
- Proxy ARP
- ICMP Redirects
- Unnumbered Interfaces
- Verification Commands
- IPv4 Configuration Labs
