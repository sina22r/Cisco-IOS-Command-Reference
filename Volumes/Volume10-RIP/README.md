## About this Volume

This volume covers the configuration, verification, and
troubleshooting of the Routing Information Protocol (RIP) on
Cisco IOS.

RIP is a distance-vector routing protocol that uses hop count
as its routing metric. Although rarely deployed in modern
enterprise environments, it remains an important protocol for
understanding routing fundamentals and CCNA/CCNP concepts.
