## About this Volume

This volume covers the Enhanced Interior Gateway Routing
Protocol (EIGRP) for IPv4 and IPv6 on Cisco IOS.

EIGRP is an advanced distance-vector routing protocol that
uses the Diffusing Update Algorithm (DUAL) to provide fast
convergence, loop-free routing, unequal-cost load balancing,
and efficient route calculation.

This volume covers both Classic Mode and Named Mode
configuration used in modern Cisco IOS XE releases.
