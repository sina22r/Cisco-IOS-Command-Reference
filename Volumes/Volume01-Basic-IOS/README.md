# Volume 1 — Basic IOS

Welcome to **Volume 1** of the Cisco IOS Command Reference.

This volume introduces the Cisco IOS Command-Line Interface (CLI) and the fundamental commands required to configure, manage, and troubleshoot Cisco routers and switches.

It is designed for:

- CCNA candidates
- CCNP ENCOR candidates
- Network Engineers
- System Administrators
- Cisco beginners

---

# Learning Objectives

After completing this volume, you will be able to:

- Navigate the Cisco IOS CLI
- Understand CLI modes
- Configure basic device settings
- Save and restore configurations
- Verify device information
- Troubleshoot common issues
- Use verification commands effectively
